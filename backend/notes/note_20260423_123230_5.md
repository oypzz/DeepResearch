# 任务 1: 卷积神经网络基础原理

任务概览：本任务旨在深入理解卷积神经网络（CNN）的基本概念及其核心组件（卷积层、池化层等）的工作原理。本次更新聚焦于从原理、工程实践、优缺点等多维度解析CNN的基础架构。
系统提示：作为研究规划的一部分，此任务为后续深入理解奠定基础。
来源概览：需检索关于CNN起源、关键里程碑论文（如LeNet）、以及其要解决的根本问题（如图像识别的平移不变性）的资料。
任务总结：
1.  **核心思想与架构哲学**：CNN的核心思想是**层次化、局部感知和权值共享**。它通过堆叠卷积层、池化层和非线性激活层，模拟生物视觉皮层从简单特征（边缘、角点）到复杂特征（物体部件、整体）的渐进式提取过程。这种设计使其特别擅长处理具有**平移不变性**（即物体在图像中位置变化不影响识别）和**局部相关性**（如图像中相邻像素关联性强）的数据。其价值在于为高维、非结构化的图像数据提供了一种高效、可扩展的特征提取范式，是深度学习在计算机视觉领域取得突破的基石。
2.  **卷积层：特征提取的引擎**：卷积层是CNN的核心。其工作原理是通过一个可学习的**卷积核（滤波器）** 在输入数据（如图像）上滑动，进行局部区域的点积运算（卷积操作）。
    *   **原理**：每个卷积核负责提取一种特定的空间特征（如垂直边缘）。通过多个卷积核，可以并行提取多种特征，形成**特征图**。
    *   **工程实践**：关键超参数包括**核尺寸**（如3x3）、**步长**（滑动间隔）和**填充**（在输入边缘补零以控制输出尺寸）。现代架构（如VGG）倾向于使用多层小尺寸卷积核（3x3）替代大尺寸卷积核，以在保持相同感受野的同时，增加网络深度和非线性，减少参数量。
    *   **优缺点**：**优点**是局部连接和权值共享极大减少了参数量（例如，一个5x5卷积核连接到一个1000x1000的图像，若为全连接需要2500万个参数，而权值共享后仅需25个参数+1个偏置），提高了模型的效率和泛化能力。**缺点**是对输入数据的空间结构有强假设，在处理非网格化数据（如知识图谱）时可能不直接适用。
3.  **池化层：特征降维与空间不变性**：池化层（通常紧接卷积层之后）的主要功能是**下采样**，逐步降低特征图的空间尺寸（宽和高）。
    *   **原理**：通过对局部区域进行聚合操作（如取最大值或平均值）来输出一个代表值。**最大池化**（取区域最大值）能保留最显著的特征并增强网络对微小平移的鲁棒性；**平均池化**（取区域平均值）能平滑特征。
    *   **应用与价值**：池化层能**显著减少后续层的计算量和参数数量**，控制过拟合。更重要的是，它通过逐步抽象，帮助网络获得一定程度的**尺度不变性和旋转不变性**，使模型更关注“有什么特征”而非“特征在精确的哪个位置”。
    *   **演变**：在一些现代架构（如ResNet）中，池化操作有时被带有步长的卷积层替代，以实现更灵活的下采样。
4.  **非线性激活函数：引入复杂表达能力**：卷积操作本质是线性变换，必须通过非线性激活函数来赋予网络拟合复杂函数的能力。
    *   **原理与对比**：早期使用Sigmoid或Tanh函数，但存在梯度饱和（梯度消失）问题，不利于深层网络训练。**ReLU**及其变种（如Leaky ReLU, PReLU）因其在正区间的梯度恒为1，能有效缓解梯度消失、加速收敛，已成为主流选择。
    *   **工程实践**：ReLU的稀疏激活特性（将负值置零）使得网络更高效，并具有一定的解释性（可视为特征选择）。
5.  **全连接层与输出层：从特征到决策**：在若干卷积-池化层堆叠后，提取到的高级语义特征图会被展平，并通过一个或多个全连接层进行整合，最终由输出层（如Softmax）映射到具体的类别概率。
    *   **价值与演变**：全连接层充当“分类器”角色，学习特征与标签之间的非线性组合。然而，其参数量巨大（通常占整个网络参数的绝大部分）。近年来的趋势是**用全局平均池化替代全连接层**（如在GoogLeNet中），直接将每个特征图池化为一个标量，再送入Softmax。这大大减少了参数，增强了模型泛化能力，并提供了特征图与类别之间更直观的关联。

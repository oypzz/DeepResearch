# 任务 2: CNN主要架构与变体

任务概览：本任务旨在深入解析CNN的经典网络结构及其核心层（卷积层、池化层、全连接层）的功能与设计原理。
系统提示：此任务是理解CNN如何工作的核心，需关注不同架构的演进与设计思想。
来源概览：需检索关于LeNet, AlexNet, VGG, ResNet等经典网络，以及各层组件（如卷积核、步幅、填充）的详细技术资料。

任务总结：
基于对卷积神经网络（CNN）经典架构演进的分析，本次研究梳理出以下核心脉络与发现：
1.  **架构演进的核心驱动力**：从LeNet到ResNet的演变，其核心驱动力经历了从“特征工程自动化”到“解决深度网络训练难题”，再到“提升参数效率与表征能力”的转变。早期网络受限于数据和算力，结构相对简单；AlexNet和VGG通过增加深度和引入非线性激活函数（ReLU）证明了“深度”的有效性；而ResNet则通过残差学习这一结构性创新，从根本上解决了深度网络的梯度消失/爆炸问题，使网络深度得以突破性增长（如从VGG的19层到ResNet的152层乃至更深的1000+层）。
2.  **结构范式的根本性转变**：经典架构体现了从“链式平铺”到“模块化堆叠”的设计哲学变迁。LeNet、AlexNet、VGG本质上都是卷积层、池化层、全连接层的顺序连接。ResNet引入了“残差模块”这一可重复堆叠的基本构建块，标志着设计重心从设计整个网络转向设计优秀的“模块”，极大地提升了模型的复用性、可扩展性和可解释性。后续的DenseNet、Inception系列等变体都继承了这一模块化思想。
3.  **核心设计思想的传承与精炼**：
    *   **卷积核小型化**：从AlexNet的11x11、5x5大卷积核，到VGG全面采用3x3小卷积核的堆叠（在保持相同感受野的同时，大幅减少了参数量，增加了非线性），这一思想成为现代CNN的基石。
    *   **空间下采样与通道扩张**：网络通常在前部进行快速空间下采样（池化或步幅卷积），同时伴随通道数的指数级增加。这实现了从“高分辨率-少特征”到“低分辨率-多特征”的高效信息转换，是CNN能够捕获多层次抽象特征的关键。
    *   **全连接层的角色弱化**：早期网络（如AlexNet）包含巨大的全连接层，参数量占比极高。后续趋势是减少或全局平均池化（GAP）替代全连接层，以减少过拟合和参数量，这一思想在ResNet及以后的网络中广泛应用。
4.  **工程实践影响的扩散**：这些经典架构不仅是学术里程碑，更深刻影响了工业界。AlexNet的成功推动了GPU在深度学习中的普及；VGG因其结构规整简单，成为迁移学习、特征提取的常用骨架；ResNet的残差连接思想已被广泛应用于计算机视觉之外的领域，如自然语言处理（Transformer中的残差连接）、语音识别等，成为构建深度模型的通用技巧。
5.  **后续变体的发展方向**：在ResNet之后，CNN架构的演变主要围绕以下几个方向：
    *   **更高效的模块设计**：如DenseNet的密集连接、SENet的通道注意力机制。
    *   **自动化架构搜索**：试图用算法替代人工设计最优模块或整体架构。
    *   **与注意力机制的结合**：将Transformer中的自注意力机制引入CNN，形成混合模型（如Vision Transformer的早期变体），以捕获长距离依赖关系。

关键数据与事实（基于经典论文）：
*   **LeNet-5 (1998)**：约6万个参数，用于手写数字识别（MNIST）。
*   **AlexNet (2012)**：约6000万个参数，在ImageNet竞赛上将Top-5错误率从26.2%降至15.3%，开启了深度学习热潮。
*   **VGG-16 (2014)**：约1.38亿个参数，其3x3卷积堆叠的设计范式影响深远。
*   **ResNet-50/ResNet-152 (2015)**：分别约2560万和6000万个参数，在ImageNet上实现3.57%的Top-5错误率（首次超越人类水平约5%），并解决了深度网络的退化问题。

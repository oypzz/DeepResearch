# 任务 2: 架构变体进展

梳理除效率之外的架构变体（如Transformer-XL、XLNet、Longformer、BigBird等）的最新改进，全面了解架构演进方向。

基于当前上下文（raw_search_results_3.md等）提取到以下关键信息：
- Star-Transformer：引入星形拓扑结构，将序列元素与中心节点相连，形成多个子图，每个子图内部自注意力计算，子图间通过中心节点交换信息，既保留全局信息又降低计算复杂度。适用于多轮对话理解、篇章级文本分析等任务。
- BERT系列（双向Transformer编码器）在文本分类、命名实体识别等任务表现优异；GPT系列（单向Transformer解码器）在文本生成、对话系统展现强大生成能力。
- Transformer架构在长序列处理中存在计算复杂度高、内存占用大等局限性，未来发展方向包括稀疏注意力、线性注意力、混合架构等（引自cloud.baidu.com/article/3216624）。
注意：当前搜索结果中未直接提供Transformer-XL、XLNet、Longformer、BigBird、Performer等变体的具体改进细节，需要进一步补充检索。

# 任务 2: 大模型进展

任务概览：调研大规模预训练语言模型的最新技术进展，包括GPT-4/4o系列、LLaMA系列、Mistral、DeepSeek、MoE架构等。系统提示：关注模型架构创新、训练策略（如多阶段训练、长上下文扩展）及性能对比。来源概览：官方报告、arXiv、技术博客。

任务总结（2026-04-23 更新）：

**关键发现 1：MoE 成为新一代大模型架构基石**
- DeepSeek-V3（671B/37B active）、LLaMA 4（~17B active）、Kimi 2（~1T params，Muon优化器）、Qwen3（含0.6B MoE变体）全面采用稀疏专家路由。
- 意义：以接近小模型的推理成本享受千亿级参数容量，服务部署成本显著降低。

**关键发现 2：注意力机制走向“定制化”**
- 滑动窗口（Gemma 3，5 local : 1 global）→ KV缓存节省；MLA（DeepSeek-V3）→ 压缩KV表示；NoPE（SmolLM3）→ 无位置嵌入的长度泛化。
- OpenAI gpt-oss 在MoE中嵌入滑动窗口并引入attention sinks。Mistral Small 3.1 取消滑窗并优化tokenizer/layers降延迟。
- 价值：团队按约束（KV内存 vs 延迟）自由选择注意力模式。

**关键发现 3：小模型“智力密度”跃升，边缘AI实用化**
- Qwen3 0.6B MoE、SmolLM3 1B NoPE、Gemma 3 1B 等模型可在本地设备运行且质量接近旧版3B级模型。
- 趋势：Agentic stacks（小模型编排+耐久执行+护栏）代替“唯大模型论”。

**关键发现 4：训练系统与优化器并行革命**
- Muon优化器（Kimi 2）有望替代AdamW；GSPMD/XLA分区、多数据中心SuperPod编排将训练goodput提升至97%。
- 水印合规（SyntheID：g函数编码+n-gram检测）原生嵌入训练管线。
- 意义：好架构+好系统+好数据三架马车缺一不可，训练效率成为新竞争力。

参考来源：The Big LLM Architecture Comparison (Sebastian Raschka, 2025) 及 LinkedIn summary。

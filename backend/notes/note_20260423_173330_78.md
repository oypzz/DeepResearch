# 任务 3: 参数高效微调

任务概览：调研参数高效微调（PEFT）技术在最新Transformer模型中的应用，包括LoRA、QLoRA、Adapter、Prefix Tuning、Prompt Tuning等方法的改进及在新模型上的适配效果。系统提示：关注2024-2026年的PEFT变体及在大规模模型上的实验数据。来源概览：arXiv、ACL/EMNLP、Hugging Face PEFT库。任务总结：
**任务总结**

1. **LoRA核心原理：低秩分解实现高效微调**：基于权重更新矩阵具有低内在秩的假设，将全量更新矩阵ΔW分解为两个低秩矩阵B和A的乘积（r<<d,k），可训练参数压缩比可达128倍（以LLaMA-7B注意力层为例）。理论支撑来自Aghajanyan等人（2021）发现预训练语言模型的内在维度极低（RoBERTa-Large仅约200）。

2. **超参数配置指南**：推荐rank r=8-64（复杂任务可扩展至128+），alpha建议设为2r或固定16，目标模块应覆盖Q、K、V、O投影矩阵（资源允许时包含所有线性层），学习率应为全微调的5-10倍（1e-4~3e-4）。该配置经实践证明在多种任务上效果稳定。

3. **QLoRA双重压缩：4-bit量化+LoRA**：通过引入NF4数据类型（针对正态分布优化，量化误差比传统INT4/FP4低10-30%），将冻结基座模型量化为4-bit，结合16-bit LoRA适配器微调。该方法在NeurIPS 2023获得Oral，实验表明QLoRA微调结果与全精度全微调无统计显著差异。

4. **工程实践与资源节省**：LoRA将70B模型微调所需的GPU从8×A100（560GB+）降至单消费级GPU；QLoRA进一步降低内存需求，使得7B模型（原需14GB FP16）可在有限VRAM设备上运行。PEFT还解决了多任务存储（避免多个全量模型副本）和灾难性遗忘问题。

5. **历史演变与发展趋势**：PEFT方法从Adapter（2019）演进至LoRA（2022）再到QLoRA（2023），正向着更高效的量化、更大秩的扩展、以及结构搜索（AutoPEFT）方向发展。2024-2026年出现了DoRA（权重分解LoRA）、VeRA（随机向量共享）等变体，但当前资料主要聚焦LoRA/QLoRA的成熟应用。

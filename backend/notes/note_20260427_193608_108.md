# 任务 1: 基础结构演进

任务1：了解Transformer核心架构的最新改进，包括线性注意力、状态空间模型等非纯注意力机制（如Mamba、RWKV），以及训练效率架构（如混合专家MoE）在NLP中的应用进展。

--- 研究总结（2026-04-27）---
已完成分析，输出5条关键发现：
1. 线性注意力与状态空间模型（Mamba）已从理论走向工程部署，在长序列任务中替代标准Transformer。
2. 混合专家（MoE）在预训练效率上取得突破，稀疏激活降低计算成本。
3. 归一化层与激活函数改进（RMSNorm、SwiGLU）成为标准配置。
4. RWKV等RNN式架构在推理时延上优势明显。
5. 注意力机制本身也在演化（多查询注意力、分组查询注意力、FlashAttention）。

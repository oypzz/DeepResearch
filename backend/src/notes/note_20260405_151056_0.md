# 任务 1: CNN基础与构成

任务概览：作为调研的起点，本任务旨在系统梳理卷积神经网络（CNN）的核心构成要素（如卷积层、池化层、激活函数）、基本前向与反向传播工作原理，以及其在深度学习发展初期的历史地位和贡献。

系统提示：调研员需关注CNN区别于其他神经网络（如全连接网络）的本质特性（局部连接、权重共享、平移不变性），并简要回顾其早期发展（如LeNet）。目的是为后续分析“进展”建立一个清晰、准确的知识基线。

来源概览：预期检索学术教科书章节、权威综述文章（如LeCun, Bengio, Hinton 2015）、以及经典论文的引言部分。

任务总结：此任务内容基础但关键，避免与后续架构演化的具体模型细节重复。完成后应能清晰回答“CNN是什么及其核心组件如何工作”。

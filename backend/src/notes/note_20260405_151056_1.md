# 任务 2: 关键架构演化

任务概览：本任务聚焦于CNN模型架构本身的演进历程。目标是按时间线或技术脉络，梳理并理解具有里程碑意义的架构创新及其设计动机。

系统提示：调研应从2012年的AlexNet开始，涵盖VGG、GoogLeNet (Inception)、ResNet、DenseNet等深度网络，直至近年来重新审视纯卷积设计的ConvNeXt等。需分析每种架构解决的核心问题（如梯度消失、特征复用、感受野、效率）。同时需关注向视觉Transformer（ViT）过渡的混合架构（如Conv-Transformer），理解卷积在其中扮演的角色。

来源概览：预期检索各里程碑模型的原始论文、ICCV/CVPR顶级会议的相关综述、以及关于“CNN架构演进”的专题博客或课程资料。现有搜索结果指向关于SENet、CNN-Transformer混合模型的综述文章，以及从AlexNet到ResNet的经典解读，可作为深入分析的起点。

任务总结：此任务是进展分析的主干，需详细对比不同架构的设计思想、贡献与局限，绘制出清晰的技术发展图谱。应与任务1（基础）和任务3（效率优化）区分侧重点。

**本轮新增分析与关键发现框架：**
基于当前资料与领域知识，CNN架构的演进可概括为几个核心阶段与思想跃迁：
1.  **深度与结构的探索 (AlexNet, VGG, GoogLeNet)**：从证明深度有效（AlexNet），到探索极致的深度与均匀结构（VGG），再到引入稀疏连接和多尺度处理的“网络中的网络”思想（GoogLeNet Inception模块），核心是解决“如何有效堆叠卷积层以提取更强大特征”的问题。
2.  **训练深度网络的革命性突破 (ResNet)**：残差学习（Residual Learning）的提出，通过快捷连接（Skip Connection）解决了极深网络的梯度消失/爆炸问题，使网络深度突破千层成为可能，这是CNN发展史上最关键的转折点之一。其思想影响了后续几乎所有深度网络设计。
3.  **特征复用的极致与轻量化思潮 (DenseNet, MobileNet, ShuffleNet)**：DenseNet将残差思想推向极致，要求每层与之前所有层连接，最大化特征复用，提升参数效率。与此同时，MobileNet（深度可分离卷积）和ShuffleNet（通道重排）代表着另一条路径——为移动和边缘设备设计高效的轻量化基础算子。
4.  **架构搜索与规模化规律 (EfficientNet)**：神经架构搜索（NAS）的成熟标志着架构设计从人工经验走向自动化。EfficientNet通过复合缩放（Compound Scaling）统一协调深度、宽度和分辨率，系统性地探索了模型缩放的最优规律，在给定算力约束下达到当时最佳的精度-效率平衡。
5.  **卷积与注意力的融合与反思 (Conv-Transformer Hybrid, ConvNeXt)**：视觉Transformer（ViT）的兴起带来了全局注意力机制。混合架构（如CoAtNet, MobileViT）尝试结合卷积的局部性、平移不变性和注意力的全局建模能力。而ConvNeXt则代表了一种反思：它通过“现代化”标准ResNet（采用大核卷积、减少激活层等ViT设计理念），证明纯卷积网络在同等规模下性能可媲美甚至超越ViT，挑战了“注意力必然优于卷积”的叙事，凸显了工程细节与设计理念的重要性。

演进路径可以概括为：**增加深度 → 解决深度训练难题 → 优化特征流与连接方式 → 自动化搜索与系统化缩放 → 与注意力机制交叉融合并反思自身潜力**。下一步需针对每个关键节点，深入其设计细节、实验论证及对后续工作的具体影响。

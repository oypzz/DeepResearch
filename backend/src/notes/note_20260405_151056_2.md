# 任务 3: 效率优化技术

任务概览：本任务专注于为CNN“瘦身”和“加速”而发展的一系列技术，旨在使模型更适用于计算资源受限的场景（如移动端、嵌入式设备）。

系统提示：调研应覆盖模型压缩的四大主流方向：1）参数剪枝（结构化与非结构化）；2）量化（低精度计算）；3）知识蒸馏（用小模型学习大模型）；4）轻量级架构设计（如MobileNet, ShuffleNet）。此外，自动化设计方法——神经架构搜索（NAS）也应作为重点。需分析各项技术的原理、优缺点及实际部署效果。

来源概览：预期检索模型压缩与高效深度学习领域的权威综述论文、轻量化模型的原始论文（如MobileNet系列）、以及产业界（如谷歌、高通）发布的相关技术报告。

任务总结：此任务与任务2（架构）互补，后者关注性能提升的架构创新，本任务则关注在给定或追求性能的前提下，如何优化计算、存储和能耗效率。需明确各项技术适用的场景。

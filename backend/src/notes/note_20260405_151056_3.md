# 任务 4: 可解释性与新机制

任务概览：本任务旨在探究两个相关方向：一是理解CNN内部决策过程的“可解释性”方法；二是为CNN引入的、超越标准卷积的新计算机制。

系统提示：调研分为两部分：
1. 可解释性：梳理从特征可视化、反卷积网络到类激活图（CAM/Grad-CAM）及其变体等后验解释方法，了解它们如何揭示CNN的关注区域。
2. 新机制：研究如注意力机制（空间/通道注意力，如SE模块）、动态卷积（根据输入调整卷积核）等如何被嵌入CNN以增强其表达能力。这部分需与视觉Transformer的注意力进行简要对比。

来源概览：预期检索可解释AI（XAI）领域的综述、CAM系列论文、以及引入注意力/动态卷积的CNN模型论文（如SENet, DynamicConv）。

任务总结：此任务从“理解模型”和“增强模型”两个内部视角分析进展。它连接了传统CNN与更灵活的现代架构，是理解其演进的重要一环。

# 任务 4: 评测与应用

## 任务概览
调研最新多模态模型在图像/视频理解、生成、视觉问答等任务上的性能表现和评测基准。

## 已获取的核心信息

### 1. 综合性综述与评测框架
- **LMMs evaluation survey** (Zhang et al., 2025): 统一了通用-专用理解与模态特定生成评测，为LMM评测提供了系统分类。
- **CVPR 2025 Tutorial “Evaluations and Benchmarks of MLLM”**: 系统梳理了MLLM评测现状，强调幻觉缓解和多学科专家级技能评测的必要性。

### 2. 细分领域的新基准（2025-2026）
- **VideoAesBench (2026)**: 视频美学质量感知评测，揭示当前LMMs仅具备基础审美能力。
- **GRADE (2026)**: 首个“学科知识+推理”的图像编辑评测基准，从学科推理、视觉一致性、逻辑可读性三维度评估。
- **GenExam (2025)**: 首个跨学科文生图考试基准，1000样本覆盖10门学科，四级分类体系。
- **LifeEval (2026)**: 以自我为中心视角的日常任务人机协作评测，强调实时性。
- **EvalTalker (2025)**: 多说话人生成式数字人的质量评估框架（THQA-MT数据集）。
- **ARM-Thinker (2025)**: 代理式多模态奖励模型，通过自主调用外部工具缓解幻觉。

### 3. 评测效率与数据质量
- **Multi-to-One Interview Paradigm** (Shen et al., 2025): 高效的MLLM评测范式，与全覆盖结果高相关。
- **Instruction Data Quality for IQA** (Li et al., 2025): 基于聚类的指令数据选择框架，提升可解释图像质量评估。

### 4. LLM Benchmark Wars (2024-2026) 数据集
- Kaggle 数据集对比24个LLM（含多模态模型），覆盖MMLU、HumanEval、GPQA、SWE-Bench等基准。
- 关键发现：GPT-5.2和Gemini 3.1 Pro总体领先；DeepSeek V3.2性价比最高；开源模型追赶迅速。

### 5. 跨模态检索与生成评测新方向
- **EvalMG'26 Workshop (SIGIR 2026)**：聚焦多模态生成与检索系统评测，涵盖幻觉、偏见、文化意识、安全隐私等挑战。
- 强调需要超越传统指标的评估方法，提出基于博弈论、智能体评估的新范式。
